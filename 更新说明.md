# 🥚 零鸡蛋小窝 · v2 更新说明

这一版加了你要的两个功能：**拖动排序** 和 **充值需要琉Q确认**。下面讲清楚怎么用、怎么更新上线。

---

## ✨ 新功能一：拖动排序

- 进入**管理员模式**后（见下方），每张小伙伴卡片右上角会出现一个 `⠿` 小把手。
- 按住把手上下拖动，就能调整小伙伴的顺序，松手后会**自动保存到云端**，其他人刷新也会看到新顺序。
- 手机和电脑都能拖。没进管理员模式时看不到把手，普通人不会误改顺序。

## ✨ 新功能二：充值要琉Q确认才到账

现在分两种身份：

| 身份 | 能做什么 |
|---|---|
| **普通人**（默认） | 查看、登记领取、**申请充值** |
| **琉Q（管理员）** | 上面全部 + **确认/拒绝充值**、直接充值、拖动排序、删除记录、添加小伙伴 |

**充值流程变成这样：**

1. 任何人点某个小伙伴的「💰 申请充值」，填金额后提交 —— 这时**余额不会马上加**，而是进入「⏳ 待确认」列表。
2. 琉Q 在「⏳ 待确认」标签里看到这条申请，点「✅ 确认到账」后，余额才真正加上去并记入历史；或者点「❌ 拒绝」把它划掉。
3. 琉Q 自己在管理员模式下充值是**直接到账**的，不用再确认一遍。

> 顶部统计里的「剩余 / 累计充值」只统计**已确认**的，待确认的不会先算进去，不会算错账。

## 🔑 管理员怎么登录（重点）

网站没有账号系统，所以用一个 **管理密码（PIN）** 来区分琉Q和其他人。

- 点页面顶部的 **「🔑 管理员登录」**，输入密码即可。
- 登录后这台手机/电脑会**记住身份**，下次打开自动是管理员（除非清了浏览器缓存或点了退出）。
- **默认密码是 `1688`**。

### ⚠️ 强烈建议改成你自己的密码

默认密码大家都可能知道，建议改掉。改的方法（不用改代码）：

1. 打开 Cloudflare → 进入你的 Pages 项目。
2. 左边 **Settings（设置）→ Environment variables（环境变量）**。
3. 点 **Add variable**，名字填 `ADMIN_PIN`，值填你想要的密码（比如 `520131`），保存。
4. 回到 **Deployments**，点最近一次部署右边的 **⋯ → Retry deployment**（重新部署一次让密码生效）。

改完之后，旧的 `1688` 就失效了，只有你设的新密码能登录。

## 🚀 怎么更新到线上

你现在的网站已经在跑了，更新只要替换两样东西：

1. 解压本压缩包，得到 `index.html` 和 `functions/` 文件夹。
2. 把它们传到你的 GitHub 仓库，**覆盖**原来的 `index.html` 和 `functions/` 文件夹（整个 functions 文件夹一起替换最稳妥）。
3. 提交（commit）后，Cloudflare 会**自动重新部署**，等 1～2 分钟，刷新网页就能看到更新。

### 关于你的老数据

放心，**原来的小伙伴、余额、历史记录都不会丢**。新代码会自动兼容老数据，只是多加了一个「待确认」列表而已，不会动到已有的账目。

## ❓ 常见问题

**Q：我忘了密码怎么办？**
A：去 Cloudflare 的环境变量里看 `ADMIN_PIN` 的值；如果没设过，那就是默认的 `1688`。

**Q：别人能不能绕过确认直接加余额？**
A：不能。确认、直接充值这些操作，后端都会校验密码，光改网页是没用的。

**Q：拖动排序别人能改吗？**
A：只有登录管理员的人能拖，普通人看不到拖动把手。
